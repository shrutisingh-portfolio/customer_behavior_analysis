📊 Customer Behavior Analysis Dashboard

🔍 Overview

This project analyzes customer shopping behavior to uncover actionable insights for business decision-making. It demonstrates an end-to-end data analytics workflow — from data processing to visualization and reporting.

📁 Dataset

The dataset includes:
Customer demographics (Gender, Location)
Purchase details (Item, Amount, Frequency)
Payment methods
Review ratings
Subscription status

🛠️ Tech Stack

Python (Pandas, NumPy, Matplotlib, Seaborn)
PostgreSQL (SQL Queries & Analysis)
Power BI (Dashboard Visualization)
Gamma (Presentation Creation)
Jupyter Notebook

⚙️ Workflow

🔹 Data Loading & Cleaning
Loaded CSV dataset using Pandas
Handled missing values & duplicates
Standardized formats and data types

🔹 Exploratory Data Analysis (EDA)
Customer distribution analysis
Purchase behavior trends
Visualizations using charts and graphs

🔹 SQL Analysis (PostgreSQL)

Imported cleaned data into PostgreSQL
Performed queries to analyze:
Revenue by category
Customer segments
Purchase patterns

🔹 Dashboard (Power BI)

Built an interactive dashboard with:

📌 Total Customers
💰 Average Purchase Amount
⭐ Average Review Rating
📊 Revenue & Sales by Category
🎯 Customer Segmentation
📊 Dashboard Preview

📈 Key Insights

Identified top-performing product categories
Analyzed customer segments contributing to revenue
Observed trends in purchase frequency and spending
Evaluated impact of subscription status on sales
